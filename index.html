What is Github ?

Github is a Version Control System

  • Easily recover files
  • Roll back to previously working state
Steps Command Meaning
Step 1 git clone your github link To clone git repo in your local computer
Step 2 git status to check wheather the directory is git repositary or not
Step 3 git add file name To add specific file in ur git repo
Step 4 git add . To add all the files in ur git repo
Step 5 git commit -m "your message" To commit what we have done changes in the file or repo
Step 6 git push To push all the files to github
